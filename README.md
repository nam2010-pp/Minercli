
# 💻 DUCO CLI Miner

Đào [Duino-Coin (DUCO)](https://duinocoin.com/) bằng Node.js **hoàn toàn không cần giao diện web**.

## ⚙️ Tính năng
- ✅ CLI miner (không web)
- 🧪 Kiểm tra SHA1 trước khi đào
- 🛠 Dùng được trên GitHub Actions, VPS, Termux

## 🚀 Sử dụng
```bash
npm i
node cli.js
```
